Place your JPG photo in this folder and name it `photo.jpg`.

Recommended steps:
1. Save your image as `assets/photo.jpg` (JPG or JPEG format).
2. The hero in `index.html` references this path and will display the image.
3. Alt text used in `index.html` is: "Pavan — portrait". Update the `alt` attribute if you want a different description.

If you uploaded the photo via the editor attachments, move or rename it to `assets/photo.jpg` so the page can display it.
